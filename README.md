# simulador_bonus_is
Um simulador de bônus que criei na época em que trabalhei na InvestSmart.

Esse simulador foi criado quando percebi que os liderados de outras áreas não tinham acesso fácil à informação. Como os dados ficavam restritos ao Power BI, o acesso era difícil, já que nem todos possuíam a licença. Aí, pensando em democratizar a informação que era um dos objetivos da minha área, criei esse simulador simples em HTML, que foi indexado no hub. Para isso, aproveitei a estrutura do algoritmo que eu já tinha construído em DAX, precisando apenas adaptar para a linguagem HTML.

Neste repositório, tentei recriar o passo a passo do projeto, commitando as atualizações que fiz na época.

A bonus simulator I created during my time at InvestSmart.

I built this simulator when I realized that team members from other departments didn't have easy access to this information. Since the data was restricted to Power BI, access was difficult, as not everyone had a license. So, wanting to democratize information which was one of our department's main goals, I created this simple HTML simulator and indexed it on the hub. To do this, I leveraged the algorithm structure I had already built in DAX, so I only needed to adapt it to HTML.

In this repository, I tried to recreate the step-by-step process of the project, committing the updates I made back then.
